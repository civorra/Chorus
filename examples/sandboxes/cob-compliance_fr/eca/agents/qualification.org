#+AGENT: qualification
#+PIPELINE_POS: 1
#+RULES_DIR: rules/qualification

* Domaine
  Vérification des exigences matériaux bois (§2) :
  classe de résistance minimale (NF EN 338), taux d'humidité (§2.2),
  flèche admissible (§2.3 — EC5 §7.2).
  Cet agent qualifie chaque élément bois et pose le slot de présence
  pour l'agent ossature (éléments structuraux) et pour thermique
  (éléments non structuraux : isolants, pare-vapeur).

* Slots de ciblage
  Stratégie B — slot de présence.
  Slot ciblé : =besoin_qualification=
  Posé par : le feed initial, sur chaque Frame créé.

  | Slot                 | Posé par | Valeur attendue |
  |----------------------+----------+-----------------|
  | besoin_qualification | feed     | 1               |

  Contrat pré-population (feed) :
  Chaque Frame doit porter à minima :
    - =besoin_qualification= = 1
    - =type_element= : l'un de montant_porteur, montant_non_porteur,
      lisse_basse, lisse_haute, entrait, chevron, isolant_laine,
      isolant_rigide, membrane_etanche
    - =classe_bois= (C14..C40) — pour les éléments bois
    - =humidite_pct= — taux d'humidité mesuré (éléments bois)
    - =fleche_mesuree_mm= — flèche constatée (éléments bois)
    - =portee_mm= — portée en mm (éléments bois)

* Pipeline E/S
  Entrants : =besoin_qualification=, =type_element=, =classe_bois=,
             =humidite_pct=, =fleche_mesuree_mm=, =portee_mm=
  Sortants : =qualifie= ('OUI'/'NON'), =motif_refus= (si NON),
             =besoin_ossature= (1 — types structuraux qualifiés),
             =besoin_thermique= (1 — isolants/membranes)

* Ontologie
  - Classe de résistance bois (NF EN 338) : C14 < C16 < C18 < C24 < C30 < C35 < C40
  - Plage humidité admissible : 12–18 % (§2.2)
  - Flèche admissible : L/300 (§2.3 — EC5 §7.2)
  - Types structuraux (ossature) : montant_porteur, montant_non_porteur,
    lisse_basse, lisse_haute, entrait, chevron
  - Types non structuraux (thermique directement) : isolant_laine,
    isolant_rigide, membrane_etanche

* Catalogue des règles
  R01-classe-resistance  — vérifie que la classe bois ≥ classe minimale pour le type (§2.1)
  R02-humidite           — vérifie 12 % ≤ humidite_pct ≤ 18 % (§2.2)
  R03-fleche             — vérifie fleche_mesuree_mm ≤ portee_mm / 300 (§2.3)
  R04-pose-besoin-ossature — pose besoin_ossature=1 sur éléments bois qualifiés structuraux
  R05-pose-besoin-thermique — pose besoin_thermique=1 sur isolants/membranes

* Helpers Perl
  ** classe_min_pour_type($type_element) → string (classe minimale)
     Source corpus : §2.1 — Classes de résistance par type d'usage (NF EN 338)
     Retourne la classe minimale admissible pour un type d'élément donné.
     Retourne undef pour les types non bois (isolants, membranes).

  ** classe_ok($classe_bois, $classe_min) → bool
     Source corpus : §2.1 — NF EN 338 — ordre des classes C14..C40
     Retourne 1 si $classe_bois >= $classe_min dans l'ordre de résistance.

  ** fleche_admissible_mm($portee_mm) → float
     Source corpus : §2.3 — Flèche admissible = L/300
     Retourne portee_mm / 300.

  ** type_est_structurel($type_element) → bool
     Source corpus : §1.1 — types structuraux (ossature + charpente)
     Retourne 1 si montant_porteur, montant_non_porteur, lisse_basse,
     lisse_haute, entrait, chevron.

  ** type_est_non_structural($type_element) → bool
     Source corpus : §1.1 + §4.1 — types non structuraux
     Retourne 1 si isolant_laine, isolant_rigide, membrane_etanche.

* Contraintes & Pitfalls
  - Stratégie B : EXCEPTION obligatoire sur toutes les règles (idempotence)
  - R01/R02/R03 : retourner 0 si classe_bois absent (Frame non bois)
  - R04 : ne poser besoin_ossature que si type_est_structurel()
  - R05 : ne poser besoin_thermique que si type_est_non_structural()
  - R01, R02, R03 : poser qualifie='NON' + motif_refus ET retourner 1 si KO
    (bloquer les règles suivantes via EXCEPTION sur qualifie)
  - Terminaison de cet agent : via cut() dans R04 et R05 après pose du slot
