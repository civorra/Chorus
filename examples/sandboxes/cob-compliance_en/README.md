# cob-compliance_en

A complete end-to-end example of the Chorus + AI agent chain.  
Domain: **Timber-frame construction** — BS EN 338, EC5, Building Regs Part L/B, BS EN 13501.

## Run the example

```sh
perl run.pl project-demo.json
```

## Regenerate the infrastructure from the KB

```sh
# From an AI agent, in Chorus context:
chorus-check cob-compliance_en project-demo.json
```

## Enrich the knowledge base

```sh
# Add a supplementary corpus:
chorus-feed cob-compliance_en <new-corpus.txt> --enrich
```

## Pipeline

| Pos | Agent | Slug | Specialty |
|---|---|---|---|
| 1 | `TimberFrame::Agent::QualifyMaterial` | `qualify-material` | Material qualification (class, moisture, deflection) |
| 2 | `TimberFrame::Agent::CheckGeometry` | `check-geometry` | Frame geometry (spacing, sections, splice) |
| 3 | `TimberFrame::Agent::CheckThermal` | `check-thermal` | Thermal & vapour control |
| 4 | `TimberFrame::Agent::CheckFire` | `check-fire` | Fire safety (REI, lining, plasterboard) |
| 5 | `TimberFrame::Agent::CheckCompliance` | `check-compliance` | Global compliance + pipeline termination |
